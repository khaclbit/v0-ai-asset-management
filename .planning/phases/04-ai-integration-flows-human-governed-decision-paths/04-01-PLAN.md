<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User/Operator → AI Orchestrator | Untrusted human input enters assistant/OCR/predictive decision flows |
| AI Recommendation → Human Approval Action | AI output crosses into business-impacting decision boundary |
| Orchestration Service → Audit Log Store | Security-critical decision records cross into append-only evidence layer |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | S (Spoofing) | Approval checkpoint in all AI flows | mitigate | Require role-authenticated actor identity and approver-role verification (manager/asset-admin) before approval/override per D-14; document verification points in `04-ai-approval-audit-control-model.md`. |
| T-04-02 | T (Tampering) | AI decision log chain | mitigate | Enforce append-only immutable log policy per D-16 with correlation-linked event chaining per D-13 in `04-ai-approval-audit-control-model.md`. |
| T-04-03 | R (Repudiation) | Human approval/override decisions | mitigate | Require mandatory structured payload (actor, role, decision, rationale, timestamp, affected entity, correlation ID) per D-13 and include in each workflow artifact. |
| T-04-04 | I (Information Disclosure) | Assistant response output | mitigate | Restrict assistant responses to internal asset data only (D-01), include provenance metadata (D-03), and define insufficient-data response path (D-02) to prevent unsupported disclosure/speculation. |
| T-04-05 | D (Denial of Service) | High-risk predictive backlog | mitigate | Define SLA timer and automatic escalation to asset manager on breach per D-12; include explicit escalation branch and audit event emission in predictive workflow. |
| T-04-06 | E (Elevation of Privilege) | Override and high-impact approval path | mitigate | Enforce segregation of duties (D-14) and dual-control for high-impact overrides (D-15) with explicit disallow rules for initiator self-approval in control model artifact. |
</threat_model>
