## Design decisions

- SQLAlchemy 2.0 style (`Mapped`, `mapped_column`)
- UUID primary keys (`postgresql.UUID(as_uuid=True)`, server_default gen_random_uuid())
- Python Enum classes define allowed values (DB-level enums)
- FKs: `assignee_id` on Asset uses `ondelete="SET NULL"` (asset survives user deletion)
- FKs: `asset_id` / `assignee_id` on Assignment & Maintenance use `ondelete="RESTRICT"` (cannot delete asset with records)
- Indexes on `asset_id` + `assignee_id` in Assignment and Maintenance tables
